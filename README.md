RRENG Secret Project
====================

[GUI Demo](https://ejrh.github.io/rreng/gui/)

Controls
--------

Moving around:

    WASD    - Move
    QE      - Rotate 
    ZX      - Zoom
    PgUp/Dn - Pitch

Editing:

    Left click  - Raise
    Right click - Lower
    Drag        - Flatten

Other:

    F5  - Toggle debug mode
    ESC - Exit (except in web mode)
